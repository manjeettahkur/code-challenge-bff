## Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/manjeettahkur/code-challenge-bff.git
  ```
2. Navigate to the project directory:
  ```bash
  cd code-challenge-bff
  ```
3. Install the Dev dependencies:
  ```bash
  npm install
  ```
